Desafio das Tags

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobriar qual tag deverá utilizar em cada um

    Tag <p>
  • Título
    Tag <h(1~6)>
  • Imagem
    Tag <img>
  • Marcação
    tag <mark>
  • Negrito
    Tag <strong>
  • Link
    Tag <a>
  • Lista ordenada
    Tag <ol>
